# Addon Boulangerie pour Garry's Mod DarkRP

Cet addon ajoute deux métiers de boulangers (traditionnel et industriel) avec un système de production de pains et viennoiseries.

## Installation

1. Copier le dossier `boulangerie` dans `garrysmod/addons/`.
2. Redémarrer le serveur.

## Fonctionnalités principales
- Entités de fabrication (sacs de farine, bouteilles d'eau, pâte, plan de travail, fours).
- Système de mélange, façonnage et cuisson des produits.
- Interface Derma pour gérer les stocks et les commandes.
- Persistance simple via fichiers data.

## Ajout de nouveaux produits
Consultez les fichiers dans `lua/entities` pour voir comment déclarer un produit. Dupliquez un dossier `product_` existant et adaptez les paramètres.
